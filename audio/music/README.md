# Local Music Directory

This directory stores local audio files used by the music playback module.

The audio files are not included in this repository because they may be copyrighted materials.

Users can add their own music files here.

## Supported Formats

Supported audio formats:

- MP3
- WAV


## Usage

When the child requests music during interaction, the system randomly selects an audio file from this directory and plays it through the audio output device.


Example:


audio/
└── music/
├── README.md
├── music1.mp3
└── music2.wav



## Notes

- Please ensure that you have permission to use the audio files.
- Empty directory is allowed before adding personal music files.
